【方法二】使用 Visual Studio

1. 双击打开 "GalgameLauncher.csproj"
2. 选择 "Release" 模式，平台选 "x64"
3. 右键项目 → "发布"
4. 点击"完成" → 再次点击"发布"
5. 找到 exe：bin\Release\net6.0-windows\win-x64\publish\GalgameLauncher.exe
6. 复制到 galgame-engine 目录

使用方法：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 将 GalgameLauncher.exe 放在 galgame-engine 目录
   （与 index.html 同级）

2. 双击运行 GalgameLauncher.exe

3. 程序会自动：
   - 启动 HTTP 服务器
   - 打开浏览器访问游戏页面

4. 保持黑色命令行窗口开启以维持服务器运行

5. 需要停止时，关闭窗口或按 Ctrl+C


技术原理：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

核心实现：
- 使用 .NET Framework 内置的 HttpListener 类
- 监听 localhost:8080 端口的 HTTP 请求
- 解析 URL 路径，返回对应的静态文件
- 自动检测工作目录并以此为根目录提供文件服务

为什么文件较大（60MB）？
- 使用了 SelfContained=true 模式
- 将所有.NET 运行时打包到单个 exe 中
- 优点：完全独立，无需安装任何运行时
- 适用场景：分发给最终用户
